Meus Trabalhos

Projeto Receitas

Primeiro projeto do curso onde foi usado os conceitos básicos de html, como lista ordenada e não ordenada. Também foram usados comandos básicos de CSS para estilização de backgrounds e fontes. Posteriormente o projeto foi reestilizado.

Projeto Landing Page

Neste projeto foi criada uma Landing Page para um site ficticio de rock. Foi exigida uma melhor estruturação do código html com o uso de containers. surge então um novo conceito no código CSS com a introdução do Flash Box.

Projeto Dashboard

Aqui com todos os conceitos de html consolidados, avançamos no CSS com o uso do Grid e flex em algumas ocasiões, o que permitiu mais praticidade e facilidade para ajustar o design da página no formato ideal.

Projeto Formulário

Chegou a hora de usarmos o formulário. Neste projeto foram aplicados os conceitos que abragem cada campo, bem como suas formas de validação, usando o regex. Para armazenamento dos dados, foi usada uma planilha google docs.
